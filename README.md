# Strategic Networking CRM Dashboard

A self-initiated CRM-style networking management project designed to organize professional connections, relationship-tracking, strategic outreach workflows, and networking prioritization.

## 🌐 Live Project

🔗 https://atulprasad98.github.io/Networking-CRM/

---

# 📌 Project Overview

This project was built to simulate how professionals can manage and organize valuable networking relationships using a simplified CRM-style interface.

The idea behind this project was to combine:
- Professional networking
- Relationship management
- Contact organization
- Strategic follow-up systems
- CRM-style workflow thinking

into a clean and structured digital dashboard.
